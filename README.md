# WordPress XML-RPC Lab

Script de laboratorio para practicar autenticación XML-RPC en WordPress utilizando el método `wp.getUsersBlogs`.

## Uso

```bash
chmod +x xmlrpc_bruteforce.sh
./xmlrpc_bruteforce.sh
```

## Requisitos

- curl
- wordlist rockyou.txt
- entorno de laboratorio autorizado

## Objetivo

Automatizar requests XML-RPC para practicar enumeración y autenticación en entornos CTF/labs.
